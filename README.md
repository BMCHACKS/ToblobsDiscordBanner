## File Guide
---

- **`src/kritaFiles/`**  
  Contains 3 banner versions:  
  `banner1.kra` → initial version (basic structure)  
  `banner2.kra` → intermediate version  
  `banner3.kra` → final version (most refined)

- **`src/inkscapeFiles/`**  
  `finalBanner.svg` → final scalable output  
  `imageStructuring.svg` → structure/layout reference  

- **`fontFiles/`**  
  Fonts used in the artwork. Install these for correct text rendering.

## Software Compatibility
---

| File Type | Software | Notes |
|----------|----------|------|
| `.kra` | Krita | Full editing support |
| `.svg` | Inkscape | Vector editing |
| `.ttf` | Font Installer | Required for correct fonts |
| `.png` | Any Viewer | Preview only |
